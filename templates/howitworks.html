How it works

The map and location tracker both use javaScript with google map api to found your location. Won't work right away, you have to first open "chrome://flags/#unsafely-treat-insecure-origin-as-secure", enable it, and add our site to the text box to whitelist it. Then a little relaunch button should show up, so click that. Then finally go to our site and allow location. Done :)
